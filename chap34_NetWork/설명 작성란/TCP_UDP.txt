TCP와 UDP
	네트워크 통신 프로토콜 중 두가지 주요한 프로토콜
	컴퓨터 간 데이터 통신을 가능하게 함
	TCP(Transmission Control Protocol)
		연결 지향적 프로토콜 
		데이터를 보내거나 받을 때 먼저 연결을 설정
		데이터 전송이 완료되면 연결을 종료
		데이터 손실을 최소화하기 위해 데이터 전송 상태를 확인
		재전송 및 순서 제어를 지원한다
		신뢰성이 높음
		스트림 지향 프로토콜이고 데이터를 연속된 스트림으로 전송한다
		이메일 전송, 파일 다운로드 등에 쓰인다
	UDP(User Datagram protocol)
		비연결 지향적 프로토콜
		연결 설정 없이 데이터를 전송
		데이터가 손실될 수 있고, 순서가 바뀔 수 있다
		데이터 패킷 단위로 전송되며, 각 패킷은 독립적이다
		신뢰성은 낮음
		실시간 스트리밍, 온라인 게임 DNS조회 등에 사용된다.
