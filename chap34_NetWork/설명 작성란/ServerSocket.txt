ServerSocket
	주로 서버측에서 사용한다 (서버는 생성해서 수락한 다음에 데이터를 주고 받을 수 있게 해줌)
	소켓 통신을 구현할 때 사용되는 클래스
	서버 측에서 클라이언트의 연결을 수락하고 데이터를 주고받을 수 있다
	서버를 구현하면 클라이언트가 서버에 연결할 수 있다
	서버는 클라이언트와 통신할 수 있다
	
	메서드								/*서버 포트 번호*/ 
	ServerSocket sS = new ServerSocket(   int 포트번호   );
		지정된 포트 번호로 서버 소켓을 생성
	accept() : 클라이언트의 연결을 기다리고, 클라이언트가 연결되면 
				클라이언트와 통신할 수 있는 소켓을 반환
	close() : 서버 소켓을 닫음
	

Socket
	클라이언트 측에서 사용한다
	클라이언트는 Socket 객체를 생성해 서버에 연결
	서버 IP주소와 포트 번호를 사용해서 특정 서버에 연결하는 역할
	클라이언트와 서버 간의 데이터 통신을 가능하게 한다
	
	메서드
	getInputStream() : 객체에서 입력 스트림을 얻을 때 사용한다.
	getOutputStream() : 출력 스트림을 얻을 때 사용한다.
	connect() : 클라이언트가 서버에 연결할 때 사용한다, 주어진 서버 주소와 포트번호로 서버에 연결을 시도한다.
	close() : 소켓의 객체를 닫을 때 사용한다.
